# Основные проблемы AS-IS

Именно их потом нужно будет вынести отдельным блоком под диаграммой.

### P1

Учитель вручную собирает участников.

### P2

Нет единого списка записавшихся.

### P3

Количество мест контролируется вручную.

### P4

Изменения могут потеряться в переписке.

### P5

Родитель зависит от того, передал ли ребёнок информацию.
