# UI-прототип MVP

## 1. Список экранов

### Общие

| № | Экран           | Назначение                          |
| - | --------------- | ----------------------------------- |
| 0 | Стартовый экран | Вход в систему                      |
| 1 | Вход            | Авторизация пользователя            |
| 2 | Главная         | Быстрый переход к основным разделам |

### Ученик

| № | Экран                | Назначение                    |
| - | -------------------- | ----------------------------- |
| 3 | Каталог занятий      | Просмотр доступных занятий    |
| 4 | Фильтры занятий      | Выбор предмета, класса и даты |
| 5 | Карточка занятия     | Просмотр деталей и запись     |
| 6 | Подтверждение записи | Успешная запись на занятие    |
| 7 | Мои занятия          | Список активных записей       |
| 8 | Карточка моей записи | Детали записи и отмена        |

### Учитель

| №  | Экран                  | Назначение                   |
| -- | ---------------------- | ---------------------------- |
| 9  | Мои созданные занятия  | Список занятий учителя       |
| 10 | Создание занятия       | Форма создания занятия       |
| 11 | Редактирование занятия | Изменение данных занятия     |
| 12 | Участники занятия      | Список записавшихся учеников |
| 13 | Отмена занятия         | Подтверждение отмены         |

### Уведомления

| №  | Экран       | Назначение                   |
| -- | ----------- | ---------------------------- |
| 14 | Уведомления | Список системных уведомлений |

# 2. Пользовательский поток ученика

```text
Стартовый экран
↓
Вход
↓
Главная
↓
Каталог занятий
↓
Карточка занятия
↓
Записаться
↓
Подтверждение записи
↓
Мои занятия
```

## Альтернативный поток

```text
Мои занятия
↓
Карточка моей записи
↓
Отменить запись
↓
Запись отменена
```
