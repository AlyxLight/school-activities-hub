# School Activities Hub

School Activities Hub — пет-проект системного аналитика: веб-сервис для организации школьных дополнительных занятий, консультаций, кружков и факультативов.

## Цель проекта

Показать полный аналитический цикл: от описания проблемы и требований до модели данных, API-контракта и UI-прототипа.

## Проблема

Запись на дополнительные занятия в школе часто происходит через чаты, личные сообщения или устные договорённости. Из-за этого учителям приходится вручную вести списки, ученики могут пропускать изменения, а родители не всегда получают актуальную информацию.

## Решение

Сервис позволяет:

- учителям создавать занятия и управлять участниками;
- ученикам находить занятия и записываться;
- родителям получать уведомления об изменениях.

## MVP

В первую версию входят:

- просмотр списка занятий;
- просмотр карточки занятия;
- запись ученика на занятие;
- отмена записи;
- создание занятия учителем;
- отмена занятия;
- просмотр списка участников;
- уведомления о записи и отмене.

## Роли пользователей

| Роль | Возможности |
|---|---|
| Ученик | Просмотр занятий, запись, отмена записи |

| Родитель | Получение уведомлений |
| Администратор | Просмотр общей информации о системе |

## Основной пользовательский сценарий

Учитель создаёт консультацию по английскому языку для 9 класса перед ОГЭ.
Ученик видит занятие в каталоге, открывает карточку, записывается и получает подтверждение.
Учитель видит ученика в списке участников.
Родитель получает уведомление о записи ребёнка.

## Бизнес-правила

- Ученик не может записаться на отменённое занятие.
- Ученик не может записаться дважды на одно занятие.
- Количество записей не может превышать лимит участников.
- После отмены занятия всем участникам направляется уведомление.
- Только создавший занятие учитель может изменить или отменить занятие.

## API MVP

Основные операции:

| Метод | Endpoint | Назначение |
|---|---|---|
| GET | `/activities` | Получить список занятий |
| GET | `/activities/{activityId}` | Получить карточку занятия |
| POST | `/activities` | Создать занятие |
| PUT | `/activities/{activityId}` | Изменить занятие |
| PATCH | `/activities/{activityId}/cancel` | Отменить занятие |
| POST | `/activities/{activityId}/enrollments` | Записаться на занятие |
| DELETE | `/enrollments/{enrollmentId}` | Отменить запись |
| GET | `/students/me/enrollments` | Получить мои записи |
| GET | `/activities/{activityId}/participants` | Получить участников занятия |
